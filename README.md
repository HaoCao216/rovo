# Rovo Admin

Note: At now we have only UI and without logic.
## Requirements

```
nodejs >= 6.11.2
reactjs
```

## Install

```
$ npm install
```

## Run dev mode

```
$ npm start
```

## check

```
/ -> Dashboard page
/login ->  Login page
/signup -> Signup page
/contact -> Contact page
/about -> About page
```
